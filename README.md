Real-Time Sales Dashboard - Practicing Youtube Tutorial
-------------------------------------------------------------------
This project implements an interactive real-time sales dashboard using Python, Streamlit, PostgreSQL, and Plotly.
It allows monitoring of sales data dynamically, with automatic refresh every 60 seconds or manual refresh via button.
--------------------------------------------------------------------
- Connection to PostgreSQL database via SQLAlchemy.

- Display of the 100 most recent sales records.

- Interactive charts with Plotly:

- Sales by product (bar chart).

- Sales by region (pie chart).

- Sales simulation script (simulate_sales.py) that inserts mock data into the database for testing.

------------------------------------------------------------------------
Streamlit – dashboard framework.

Plotly Express (plotly.com in Bing) – data visualization.

SQLAlchemy – database connection.

PostgreSQL – relational database.

Pandas – data manipulation.
